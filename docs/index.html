Churi12/homelab

Homelab as code

This repo defines a reproducible local Kubernetes lab: k3d creates the cluster, ArgoCD applies GitOps, Prometheus and Grafana expose health signals, and a pinned nginx demo app proves workloads sync cleanly both locally and in CI.

Stack flow

+-------------------+      +-------------------+      +----------------------+
| GitHub Actions CI | ---> | k3d + k3s cluster | ---> | ArgoCD root app      |
| bootstrap.yaml    |      | local Docker lab  |      | GitOps control plane |
+-------------------+      +-------------------+      +----------+-----------+
                                                                     |
                                                +--------------------+-------------------+
                                                |                                        |
                                      +---------v----------+                   +---------v---------+
                                      | monitoring app     |                   | demo-app          |
                                      | Prometheus+Grafana |                   | nginx welcome app |
                                      +--------------------+                   +-------------------+

Stack

Component Version Role in this lab
k3d v5.6.0 Creates the disposable local cluster inside Docker.
k3s v1.28.6-k3s1 Runs the Kubernetes control plane and worker nodes.
ArgoCD Helm chart 5.46.8 Syncs the repo state into the cluster with the app-of-apps pattern.
Prometheus + Grafana kube-prometheus-stack 55.5.0 Provides cluster metrics, dashboards, and quick health checks.
nginx demo app 1.27.0 Acts as the sample workload that proves GitOps deployment works.
